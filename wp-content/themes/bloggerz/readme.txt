Thank you for downloading the Bloggerz WordPress Theme.

The theme works on the latest WordPress version without any problem. Previous versions have not been tested. We suggest you to update your WordPress installation.

/*********************************
 		   Installation
**********************************/

	1. Automatic Installation
	   Go to WP-Admin -> Appearance -> Theme -> Add New ->Upload Theme and choose the theme zip folder. Activate the Theme. That's it

	2. Manual Installation
	   Upload the Theme files to your WordPress theme folder wp-content/themes and activate the theme in your WordPress admin panel.

/*********************************
 	Customizer - Theme Options
**********************************/

Bloggerz supports the customizer for all the theme settings.
Go to WP-Admin -> Appearance -> Customize to open the Customizer

	1. Single Page Banner
	   Here you can change the Single Post/Page
		- Banner Image
		- Heading
		- Short Description
		- Overlay Option

	2. Theme layout
    - Here you can Enable/Disable the "About Author" Widget.
	  - Here you can change the Layout of your Theme. Sidebar to Full Width.

  3. Home Page Banner
  	 You can change the Home Page Banner from Header Image.

/*********************************
    Theme Widget.
**********************************/

  1. Recent Posts Thumbnail Widget
     - Bloggerz Create a recent posts thumbnail widget that shows featured image with post title. You can add this widget from Appearance > widget.
